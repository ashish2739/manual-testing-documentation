# ShopperStack Manual Testing Project

## Project Overview
Performed manual testing on ShopperStack web application.

## Scope of Testing
- Functional Testing
- UI Testing
- Validation Testing
- Boundary Value Testing
- Negative Testing

## Modules Covered
- Registration
- Login
- Search
- Add to Cart
- Wishlist
- Checkout
- Logout

## Deliverables
- Test Case Document (Excel)
- Bug Report (If any defects found)

## Tools Used
- Manual Testing
- MS Excel

## Testing Approach
CRE Approach (Consistency, Requirement, Expectation)
